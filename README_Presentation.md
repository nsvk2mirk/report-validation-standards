## How to Use

1. **Open the Presentation**:
   - Simply double-click `Report_Validation_Standards.html`
   - Or open it in any modern web browser (Chrome, Firefox, Edge, Safari)

2. **Navigate**:
   - Use the arrow buttons at the bottom
   - Use keyboard arrows or spacebar
   - Click the dots on the right side
   - Swipe left/right on mobile devices

3. **Present**:
   - Press `F11` for fullscreen mode
   - Use presentation mode in browser if available
   - The presentation is self-contained and doesn't require internet
